NODEJS

Para ejecutar el proyecto de mongo seguir los siguientes pasos, no deben salir errores.

	-ejecutar el proyecto desde la consola de mongo 

	-ubicarse en la carpeta actual cd ../NodeJS

	-ejecutar el comando >> node server/

	-en consola saldrá el siguiente mensaje: Server is listeng on port: 8082

	-ejecutar en el navegador el siguiente  link para crear el usuario:
		http://localhost:8082/app/CreateUser

	-en el navegador se podrá visualizar la información del usuario creado con 
	la contraseña encriptada.
	
	-Acceder a la app we mediante la siguiente dirección: http://localhost:8082/
	-para acceder a la app se debe ingresar las siguientes credenciales:
		usuario: usuario@gmail.com
		pass: 123456


PHP
	-Copiar toda la carpeta llamada PHP al servidor

	-editar el archivo acces.php y poner las credenciales del motor de base de datos 
	(servidor,usuario,password,baseDeDatos). La base de datos no debe existir en MySql

	-ejecutar en el navegador el script para crear la base de datos y las tablas 
	necesarias para la app http://localhost/PHP/server/create.php

	-observar que no existan errores

	-ejecutar en el navegador el script para crear usuarios:
